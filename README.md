# ConvoHub
A chatbot application made using MERN stack and OpenAI API.

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file in server directory

`PORT` = `5000`

`MONGO_URL`

`SITE_URL`

`JWT_PRIVATE_KEY`

`OPENAI_API_KEY`

`OPENAI_ORGANIZATION`

`MAIL_EMAIL`

`MAIL_SECRET`

To run this project, you will need to add the following environment variables to your .env.local file in client directory

`VITE_CLIENT_ID` #Google login api client id

## To run backend

Go to the server directory

```bash
  cd ConvoHub/server
```

Install dependencies

```bash
  npm install
```

Start

```bash
  npm start
```

## To Start Frontend

Go to the client directory

```bash
  cd ConvoHub/client
```

Install dependencies

```bash
  npm install
```

Start

```bash
  npm run dev
```
